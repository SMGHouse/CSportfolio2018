For the second line of code, that is to control the size of the the outline of the circle. The third color is to code for the color of the outlie. The numbers follow to same rules as the code on the first line, only the code, (255,255,255) codes for white. The fourth line codes for the actual circle. The first number in the parentheses code for the x-axis where the circle is positioned. The second number codes for the y-axis poisition of the circle. The third number is the width of the circle, and the fourth is the length of the circle.
